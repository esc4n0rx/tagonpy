# Change Log

## [1.0.0] - 2025-07-18

### Added
- Syntax highlighting inicial para arquivos .tg
- Suporte para seções Imports, Funcoes, Html, Css
- Highlighting Python nas seções Imports e Funcoes
- Suporte para templates Jinja2 na seção Html
- Highlighting CSS completo na seção Css
- Auto-completion básica para brackets e quotes
- Code folding por seções
- Configuração de linguagem completa

### Features
- Reconhecimento automático de arquivos .tg
- Scopes hierárquicos para melhor compatibilidade com temas
- Suporte para comentários Python (#) e HTML (<!-- -->)
- Highlighting de expressões Jinja2 {{ }} e {% %}
- Destaque especial para keywords das seções TagonPy
